    `title` - title of your web page
    `meta elements` - useful information about your webpage
`body` - Everything that is visible to users
    `headings` - Text elements
    `p` - paragraph
    `strong` - Bold text
    `em` - Italic text
    `hr` - horizontal line
    `br` - break
    `div` - block element
